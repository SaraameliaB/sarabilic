# Sara Amelia Bilic — Digital Portfolio

A modern cybersecurity-focused digital portfolio built with Next.js and deployed on Vercel.

This website serves as my personal portfolio where I showcase:
- Projects
- Blog posts
- Professional experience
- Community involvement
- Cybersecurity interests
- CV and technical growth

## Live Website

[Visit Portfolio](https://your-portfolio.vercel.app)

## About

I’m Sara Amelia, a Junior Cybersecurity GRC Consultant at EY Belgium with a background in Cybersecurity & Computer Science from HOWEST University of Applied Sciences.

This portfolio was created as a central place to document my professional journey, technical interests, projects, hackathons, cybersecurity events, and community involvement.

## Features

- Responsive portfolio website
- Blog system using MDX
- Project showcase pages
- Cybersecurity event reflections
- Community & hackathon highlights
- CV integration
- Vercel deployment pipeline

## Technologies Used

- Next.js
- React
- TypeScript
- MDX
- Vercel
- Mapbox
- Vue.js (project-specific)
- Flask (project-specific)
